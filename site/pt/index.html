Instalar — escolha seu caminho

De zero a primeiro corte em menos de 5 minutos.

Três jeitos de entrar. O primeiro se você tem terminal, o segundo se você vive num editor de IA, o terceiro pra Linux/Windows.

🟢

Um comando no terminal

Recomendado. Funciona em qualquer shell macOS ou Linux.

curl -fsSL https://onetogregorio.github.io/sermon-cuts/install.sh | bash
🟡

Através do seu editor de IA

Claude Code, Cursor, Codex, Cline, Aider — qualquer um com acesso ao terminal.

Instala a skill Sermon Cuts (https://github.com/onetogregorio/sermon-cuts) na minha máquina. Me guia pelos passos abaixo e para se algum falhar:

1. Detecta meu sistema (macOS ou Linux). No macOS, garante que o Homebrew está instalado.
2. Instala as dependências de sistema: ffmpeg (com libass), yt-dlp, Python 3.12+.
3. Clona o repo pra ~/code/sermon-cuts (ou me pergunta onde colocar).
4. Cria um venv Python dentro do repo e roda pip install -r requirements.txt.
5. Symlinka scripts/, config/, prompts/ pra ~/.claude/skills/sermon-cuts/ e copia o SKILL.md pra lá também.
6. Me pergunta se quero transcrição melhor via Groq Whisper. Se sim, abre https://console.groq.com/keys, pede pra eu colar a chave e adiciona GROQ_API_KEY=... no ~/.env.
7. Me pergunta se quero a fonte da marca Outfit Black (https://fonts.google.com/specimen/Outfit). Se sim, baixa e instala na minha pasta de fontes.
8. Roda ./scripts/pipeline.sh doctor pra confirmar que a skill está pronta.
9. Me dá um exemplo de uma linha de como cortar meu primeiro sermão.

Reporta sucesso ou o passo que falhou no final.

Docker

Linux, Windows, ou onde pip dá problema. Sem deps no host.

docker run --rm \
  -v "$(pwd)/out:/work/memory/messages" \
  -e GROQ_API_KEY=$GROQ_API_KEY \
  ghcr.io/onetogregorio/sermon-cuts <url-do-youtube>

Como funciona

Quatro passos. Em menos de 10 minutos da URL ao upload.

  1. 1

    Joga uma URL

    Link do YouTube ou um .mp4 local. Funciona em vídeos de 30 a 60 minutos.

  2. 2

    O pipeline transcreve e analisa

    Transcrição word-level (grátis via legendas auto do YouTube ou premium via Groq Whisper) + detecção de pausas pra cortes nunca dividirem palavras.

  3. 3

    Seu editor de IA propõe 10 cortes

    Cada um com tema, hook, desenvolvimento, conclusão e score de coerência. Você escolhe quais renderizar — tipicamente 5 a 8.

  4. 4

    Cortes saem verticais

    1080×1920, com face tracking, legenda brand-style queimada, áudio a -14 LUFS. Joga direto no Reels/TikTok/Shorts.
