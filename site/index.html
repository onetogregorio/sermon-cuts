The one-prompt install

Don't read install docs. Paste this.

Open Claude Code, Cursor, Codex, or any AI editor. Paste the prompt below. Answer the questions. Done in under 5 minutes. 

Install the Sermon Cuts skill (https://github.com/onetogregorio/sermon-cuts) on this machine. Walk me through these steps and stop if anything fails:

1. Detect my OS (macOS or Linux). On macOS, make sure Homebrew is installed.
2. Install system deps: ffmpeg (with libass), yt-dlp, Python 3.12+.
3. Clone the repo to ~/code/sermon-cuts (or ask me where).
4. Create a Python venv inside the repo and pip install -r requirements.txt.
5. Symlink scripts/, config/, prompts/ into ~/.claude/skills/sermon-cuts/ and copy SKILL.md there too.
6. Ask if I want to enable better transcription via Groq Whisper. If yes, open https://console.groq.com/keys, ask me to paste the key, and add GROQ_API_KEY=... to ~/.env.
7. Ask if I want the brand font Outfit Black (https://fonts.google.com/specimen/Outfit). If yes, download it and install to my user fonts folder.
8. Run ./scripts/pipeline.sh --help to confirm the skill is wired up.
9. Tell me a one-line example of how to cut my first sermon.

Report success or the failed step at the end.

Then just say: "cut this sermon: <YouTube URL>"

What's in the box

Built for messages that deserve better than a rushed Sunday night.

▶

Smart vertical reframe

MediaPipe face tracking centers the speaker on a 1080×1920 frame with smooth 2.5-second moving-average panning. No jerky cuts.

Aa

Brand-style subtitles

Outfit Black, gold on black, 3-4 words per line, sentence case, positioned above the Reels/TikTok UI bar. No shouting caps.

✦

AI proposes, you pick

Cuts must have a hook, development, and conclusion. Can't end on "porque…" or "but…". Ranked by narrative arc.

♪

Loudness-balanced audio

pyloudnorm two-pass to -14 LUFS — the Instagram and TikTok standard. Your sermon sounds right on every platform.

⌘

One command end-to-end

./scripts/pipeline.sh <url> handles ingest, transcription, VAD pause detection, and renders cleanly.

⌥

Yours to customize

Edit two files (force_style.txt, render_defaults.yaml) and the whole pipeline takes on your brand colors and font.

How it works

Four steps. Under 10 minutes from URL to upload.

  1. 1

    Drop a URL

    YouTube link or a local .mp4. Works on 30 to 60-minute videos.

  2. 2

    Pipeline transcribes & analyzes

    Word-level transcript (free via YouTube auto-captions or premium via Groq Whisper) + voice-activity boundaries so cuts never split mid-word.

  3. 3

    Your AI editor proposes 10 cuts

    Each with a theme, hook, development, conclusion, and a coherence score. You pick which to render — typically 5 to 8.

  4. 4

    Cuts ship vertically

    1080×1920, face-tracked, branded subtitles burned in, audio at -14 LUFS. Drop into Reels/TikTok/Shorts.
