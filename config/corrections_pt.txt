# Global PT-BR transcription corrections — auto-applied by 06b_scrub_srt
#
# This file is INTENTIONALLY EMPTY by default. The project ships agnostic
# of any specific preacher, vocabulary, or theological tradition. Each
# user/community adds the patterns that recur in their OWN sermons.
#
# Format: one `wrong=right` pair per line. Empty lines and #-comments
# are skipped. Whitespace around `=` is trimmed. Replacement is
# case-sensitive substring (not whole-word) — be specific to avoid
# clobbering legitimate text.
#
# Where to add entries:
#   - HERE (global)            → a transcriber mistake your preacher makes
#                                across many sermons (a specific accent
#                                pattern, a recurring theological term).
#   - memory/messages/<slug>/corrections.txt → one-off for a single sermon
#                                (a proper noun, a specific quote).
#
# Per-slug entries are applied AFTER the global file, so they can override
# or extend it.
#
# ─── Example entries (commented out — uncomment to opt in) ──────────────
#
# Theological vocabulary your transcriber gets wrong consistently:
#   Quisto=Cristo
#   espirito santo=Espírito Santo
#
# Words your specific preacher mispronounces or YouTube auto-captions
# botches in their voice:
#   canira=canseira
#
# Concatenations from your preacher's speaking style:
#   paraa =para a
